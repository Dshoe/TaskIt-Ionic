# Taskit Ionic Client
Custom todo list and task manager.

## Run this project

### Ensure Ionic and Cordova are installed
```bash
$ sudo npm install -g ionic cordova
```

### Add CORS to CouchDB
```bash
$ sudo npm install -g add-cors-to-couchdb
$ add-cors-to-couchdb
```

### Compile and run source code
```bash
$ ionic serve
```
